Gratuity

Create a function called tip that calculates gratuity from the bill. The function has two parameters bill and percentage and returns percentage percent of the bill. The tip size varies greatly from country to country, so 10% of the total amount of a customer’s check is a more or less universal solution. Therefore, if the percentage is not specified, it is to be 10. Return only the integer part of gratuity.



Memory limit: 256 MB
Time limit: 8 seconds

Caution

You may see errors in your code or execution results due to missing context. Don’t worry about it, just write the solution and press Check.
